# Infinite Canvas Mind Map Tool 🧠🚀

A powerful, high-performance, and interactive web-based Mind Mapping tool built entirely using **Vanilla HTML, CSS, and JavaScript**. No heavy frameworks used, making it incredibly fast and lightweight.

👉 **Live Demo:** `[Insert your Netlify/Vercel Live Link Here]`

---

## ✨ Features

- **Infinite Canvas with Zoom & Pan:** Scroll to zoom in/out (0.5x - 2x) and drag the empty canvas space to pan around seamlessly.
- **Interactive Nodes:** Create new ideas instantly, drag them anywhere, and double-click to edit text dynamically.
- **Drag-to-Connect Lines:** Easily link concepts together by dragging from one node's connector dot to another.
- **Custom Node Colors:** Change individual node borders or the root node's background color on the fly using a beautiful color picker overlay.
- **LocalStorage Integration:** Save multiple mind maps with custom names directly to your browser and load or delete them via a dedicated dashboard.
- **Export Options:** Download your fully customized mind map as a high-quality **PNG Image** or a professional **PDF Document** with a single click.
- **Theme Toggle:** Switch between sleek Light Mode and a comfortable Dark Mode for late-night brainstorming sessions.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (CSS Variables for Themes), Vanilla JavaScript (ES6)
- **Graphics/Connections:** SVG (Scalable Vector Graphics)
- **External Libraries (via CDN):** - `html2canvas` (For PNG exports)
  - `jspdf` (For PDF generation)
